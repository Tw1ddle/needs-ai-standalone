How It Works

This site uses Markov chains to procedurally generate original and "random" words and names. Given a set of words as training data, the generator calculates the conditional probability of a letter appearing after the sequence of letters chosen so far.

The generator maintains an array of models that look up to "n" characters back, where "n" is the order of the highest order model specified in the settings. It starts with the highest order model, and falls back to lower order models using an approach called Katz's back-off model. If you increase the order setting, you can expect to see larger chunks of familiar words appearing in the generated output.

A Dirichlet prior is used to add a constant probability that any letter may be picked as the next letter. It acts like an additive smoothing factor, making the generated names look a bit more "random". In most cases it is best to keep this low, but it can be useful to increase this if you aren't seeing many results.

The generator generates as many words as possible before filtering and sorting them according to the tweakable criteria in the settings section. The length range limits how many characters long the words are. The start, end, includes and excludes options limit the acceptable content of the word.

The similarity field allows you to sort the generated names by their similarity to the name that you enter. The similarity of the words is determined by the Damerau-Levenshtein distance.

This generator is suitable for generating all sorts of names - be they funny names, fantasy names, Scottish names, German names, Irish names, names of towns, animal names, theological names, werewolf names, all kinds of names.
